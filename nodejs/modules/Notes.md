
## Modules

- It is piece of code, which can be reused in other files.
- it helps to improve readability and maintainability od code
- using Modules we can easily scale our application
- DRY --> Dont repeat yourself
- It is used to breakdown the complex code into smaller pieces of code.

### Differnt formats of modules used.
1. CommonJS
   - It is used by nodejs by default. unless specified in package.json file.
   - require and exports

2. ES modules
   - import and export


![1766999334874](image/Notes/1766999334874.png)

export or module.exports --> we make the code avaliable for other files
import or require --> we use the code exported from other files

### Types of modules in nodejs

1. user-defiened modules/custom modules
   which is created by develoopers

2. Built-in modules/core modules
   which is already present 

3. Third-party modules
   which is usde from third party library
